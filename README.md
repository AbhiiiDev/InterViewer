# AI Code Interview Room

Real-time collaborative coding interview platform with an AI co-interviewer.

## Apps

- \`frontend\`: Next.js (React, TypeScript, Tailwind)
- \`api-gateway\`: NestJS (REST + WebSockets, later gRPC clients)

## Getting Started

### Frontend

\`\`\bash
cd frontend
npm install
npm run dev
\`\`\`

Frontend runs on http://localhost:3000

### Backend (API Gateway)

\`\`\bash
cd api-gateway
npm install
npm run start:dev
\`\`\`

Backend runs on http://localhost:3001 (if you changed the port).
